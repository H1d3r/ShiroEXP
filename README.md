# ShiroEXP

Shiro漏洞利用工具

## Environment

`JDK 8u421` | `Intellij 2024`

> ⚠️ _使用其他JDK版本可能出现未知的错误_

## Build

自行编译打jar包时，可能会遇到**找不到或无法加载主类**的问题，经排查是因为bcprov包中的签名文件，打包后删除*.DSA、*.SF文件即可
